# examples

TinyWhale source files (`.tw`) organized by feature:

- `basics/` — Comments, arithmetic, hello world
- `control-flow/` — Match expressions, shadowing
- `errors/` — Invalid programs for error testing
- `functions/` — Function definitions and calls
- `operators/` — Arithmetic, comparison, logical, bitwise
- `records/` — Record types and instantiation
- `types/` — Lists, type bounds
